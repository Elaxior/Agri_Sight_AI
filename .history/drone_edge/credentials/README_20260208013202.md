# Firebase Credentials Directory

Place your Firebase Admin SDK service account key JSON file here.

## File Name:
`serviceAccountKey.json`

## How to Get Your Credentials:

### Step 1: Go to Firebase Console
1. Visit: https://console.firebase.google.com
2. Select your project (or create a new one)

### Step 2: Navigate to Project Settings
1. Click the gear icon ⚙️ next to "Project Overview"
2. Select "Project settings"

### Step 3: Service Accounts Tab
1. Click on the "Service accounts" tab
2. Scroll down to "Firebase Admin SDK"

### Step 4: Generate New Private Key
1. Click "Generate new private key"
2. Confirm by clicking "Generate key"
3. A JSON file will download

### Step 5: Save to This Directory
1. Rename the downloaded file to: `serviceAccountKey.json`
2. Place it in this `credentials/` folder

⚠️ **NEVER commit this file to Git!**
- The `.gitignore` file already excludes this directory
- Keep your credentials secure and private
